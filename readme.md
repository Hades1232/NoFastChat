# No Fast Chat
단타 방지기

## How to use

NoFastChat.exe 실행
(3초에 한번씩 엔터키가 활성화됨)

### Json

time.json에서 수정
```c
"time" : 엔터키가 활성화 될 시간 (최소 3초)
```
